Interface CFGDefination

A context free grammer defination.

Example: A CFG G=({S}, {a}, { S -> aS | S }, S) can be represented as:

{
 nonTerminals: ["S"],
 terminals: ["a"],
 startSymbol: ["S"],
 productions: [
   {
     symbol: ["S"],
     produces: ["a", "S"]
   },
   {
     symbol: ["S"],
     produces: ["a"]
   },
   ...
 ]
}
